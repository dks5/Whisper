Мгновенное распознавание речи по горячей клавише прямо в любое текстовое поле.

Шаг 1: Подготовка и Установка

Сначала нужно установить необходимое программное обеспечение и библиотеки. Вы можете выбрать автоматический или ручной способ.

Автоматическая установка (рекомендуется)

Этот метод самый простой и быстрый. Он автоматически выполнит следующие действия:

  • Создаст папку C:\whisper_hotkey.
  • Настроит изолированное окружение, чтобы не затрагивать систему.
  • Скачает и установит все необходимые компоненты.
  1. Откройте PowerShell (можно найти через меню "Пуск").
  2. Скопируйте весь блок кода ниже, вставьте его в окно PowerShell и нажмите Enter. 
# --- ШАГ 0: РАЗРЕШЕНИЕ ЗАПУСКА СКРИПТОВ ---
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass -Force;

# --- ШАГ 1: ПОДГОТОВКА ОКРУЖЕНИЯ ---
$dir_path = 'C:\whisper_hotkey';
if (-not (Test-Path $dir_path)) {
    Write-Host "Создаём директорию: $dir_path" -ForegroundColor Green;
    mkdir $dir_path | Out-Null;
}
Set-Location $dir_path;
if (-not (Test-Path '.\venv')) {
    Write-Host "Создаём виртуальное окружение..." -ForegroundColor Green;
    python -m venv venv;
}
Write-Host "Активация виртуального окружения..." -ForegroundColor Green;
.\venv\Scripts\Activate.ps1;

# --- ШАГ 2: УСТАНОВКА ЗАВИСИМОСТЕЙ ---
Write-Host "Обновление pip..." -ForegroundColor Green;
pip install --upgrade pip;
$use_gpu = if (Get-Command nvidia-smi -ErrorAction SilentlyContinue) { $true } else { $false };
if ($use_gpu) {
    Write-Host "Обнаружена NVIDIA GPU. Установка PyTorch с поддержкой CUDA..." -ForegroundColor Green;
    pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121;
} else {
    Write-Host "NVIDIA GPU не найдена. Установка CPU-версии PyTorch..." -ForegroundColor Yellow;
    pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu;
}
Write-Host "Установка остальных библиотек..." -ForegroundColor Green;
pip install openai-whisper sounddevice keyboard pyperclip;
Write-Host "`n✅ Установка завершена. Теперь перейдите во вкладку '2. Настройка скрипта'." -ForegroundColor Cyan;
Требуется Python!
Перед запуском скрипта убедитесь, что у вас установлен Python 3.10+ (64-bit). При установке обязательно поставьте галочку “Add Python to PATH”. Скачать Python.

Шаг 2: Настройка и генерация скрипта

Используйте эту панель, чтобы настроить скрипт под себя. Изменения в полях ниже будут автоматически обновлять код. После настройки скопируйте его и сохраните в файл C:\whisper_hotkey\hotkey_whisper.py.

Например: f9, ctrl+alt+x

Для CPU лучше `base` или `tiny`.

Чем ниже, тем чувствительнее.

Индекс (0, 1...) или имя.

После сохранения файла hotkey_whisper.py переходите к финальному шагу «Запуск и автоматизация».

Шаг 3: Запуск и автоматизация

После того как вы настроили и сохранили скрипт hotkey_whisper.py, вы можете запустить его вручную, чтобы убедиться, что всё работает.

Приступим:

  1. Откройте PowerShell.
  2. Перейдите в рабочую директорию 
    Set-Location C:\whisper_hotkey
  3. Активируйте окружение: 
    .\venv\Scripts\Activate.ps1
  4. Запустите скрипт: hotkey_whisper.py: 
    python hotkey_whisper.py
Готово! 🎉 Теперь вы можете запускать диктовку нажатием и удержанием клавиши F9
